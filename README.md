# Analytics Engineering Bootcamp

## Getting Started

### Install dbt-core

```
python3 -m venv dbt-env 
source dbt-env/bin/activate 
```

and then...

```
pip install \
  dbt-core \
  dbt-postgres \
  dbt-redshift \
  dbt-snowflake \
  dbt-bigquery \
  dbt-trino
```

### Navigate to dbt project

```
cd ae_bootcamp/
```
